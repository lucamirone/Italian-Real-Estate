# Italian Real Estate

Professional real estate advisory website for American buyers exploring the Italian property market.

## Overview

This is a static lead-generation website focused on buyer profiling, advisory process, Italian market opportunity, regional fit, property types, and legal/tax disclaimers.

## Files

- `index.html` - Website structure and page sections
- `styles.css` - Responsive visual design
- `app.js` - Card rendering, form steps, validation, and lead scoring
- `assets/` - Website image assets

## Image Requirements / Visual Direction

Additional image direction - consultant should resemble Luca Mirone:

The preview image should include a professional consultant/advisor who visually resembles Luca Mirone, especially in the face.

Use Luca Mirone's official portrait/reference image already available on the website as the facial reference:

https://lucamirone.com/luca-mirone-portrait.jpg

The consultant in the image should look like Luca Mirone:

- Similar facial structure
- Similar hair color and hairstyle
- Similar beard/stubble style if visible in the reference image
- Similar age range and professional appearance
- Similar Italian/Mediterranean look
- Dressed in an elegant business-advisory style, such as a navy blazer, white or light shirt, and refined professional attire

Important:

- Do not create a random generic consultant.
- Do not make the consultant look like a different person.
- Use Luca Mirone's portrait as the reference so the advisor in the preview image clearly feels like Luca guiding American buyers through the Italian real estate process.

Scene direction:

The image should show Luca Mirone, as the consultant/advisor, speaking with or guiding American buyers who are evaluating Italian real estate. The setting should feel elegant, premium, and trustworthy, with Italian architecture, a refined Italian property, or a scenic Italian real estate context in the background.

The image should still be optimized for social sharing:

- 1200 x 630 px
- Clear subject placement
- Luca/advisor visible enough that the face is recognizable
- Buyers and Italian real estate setting visible
- No text baked into the image unless intentionally requested later
- Professional, premium, trustworthy tone

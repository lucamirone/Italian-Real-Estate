# Italian Real Estate

Professional real estate advisory website for American buyers exploring the Italian property market.

## Overview

This is a static lead-generation website focused on buyer profiling, advisory process, Italian market opportunity, regional fit, property types, and legal/tax disclaimers.

## Files

- `index.html` - Website structure and page sections
- `styles.css` - Responsive visual design
- `app.js` - Card rendering, form steps, validation, and lead scoring
- `assets/` - Website image assets

## Local Preview

Open `index.html` directly in a browser, or serve the folder with any static web server.
